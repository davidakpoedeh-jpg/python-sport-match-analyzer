# Sports Match Analyzer

A Python application for analyzing sports matches and displaying statistics.

## Features
- Match analysis
- Team match lookup
- Data validation
- User interface

## Files
- main.py - Program entry point
- MatchAnalyzer.py - Match analysis logic
- Sports_api.py - API integration
- TeamMatches.py - Team match handling
- Validation.py - Input validation
- Storage.py - Data storage
- sports_data.json - Stored data

## Run

python main.py
